第一个demo文件用来做测试git add 和 git commit命令
为第五次commit做记录

git clone +  repo url
git add .
git commit + 文件名 + -m"描述提示信息"
git log
git reflog
git reset --hard + 版本号

该文件为了测试git branch 功能
git branch
git branch -a
git checkout + 分支名
git checkout -b + 分支名
git branch --set-upstream-to=origin/远程分支名 本地分支名
git pull 拉取远程分支
git push origin master
git push origin newbranch:newbranch //推送本地的newbranch(冒号前面的)分支到远程origin的newbranch(冒号后面的)分支(没有会自动创建)
